# iOS

本项目是我个人的 iOS 学习历程





### 面向对象

掌握基本的 Web 开发技术「HTML、CSS、JS」及基本的 Swift 语法

### 目标

理解基本的 iOS 开发套路，包括基本理念，Swift 语法、开发框架、相关工具

开发一个简单的类微博 iOS App，包含 UI 布局、动态约束、网络请求、本地缓存、性能监控、Pods 管理等

迁移与融合，与 React Native 部分功能相融合，解决性能问题
